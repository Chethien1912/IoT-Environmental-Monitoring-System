# IoT Backend Moi

Backend moi cho he thong IoT dung:

- Fastify
- SQLite (`better-sqlite3`)
- JWT + refresh token
- MQTT client (`mqtt.js`)
- Socket.IO cho realtime app

## Chay nhanh

```bash
cd backend
npm install
npm run seed
npm run dev
```

Backend mac dinh chay o `http://localhost:3000`.
